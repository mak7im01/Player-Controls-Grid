# Player Controls Grid

Аддон для PulseSync, который располагает кнопки правой части плеера — **открыть текст, очередь, скачать, качество и громкость** — в два ряда.

---

## Благодарности

За идею взята тема **[Vocaloid Miku!](https://github.com/Diramix/Vocaloid-Miku)** от [diram1x](https://github.com/Diramix) — отдельное спасибо [diram1x](https://github.com/Diramix) за реализацию кнопки настроек

---

## Как это выглядит

Вместо одной длинной горизонтальной строки кнопки выстраиваются в компактную сетку:

```
[ Текст ]  [ Скачать ]  [ Громкость ]
[ Очередь] [ Качество ] [ Настройки ]
```
